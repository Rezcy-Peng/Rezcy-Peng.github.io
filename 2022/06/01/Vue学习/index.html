methods：用于定义的函数，可以通过 return 来返回函数值

模板语法

Vue.js 使用了基于 HTML 的模板语法，允许开发者声明式地将 DOM 绑定至底层 Vue 实例的数据。
Vue.js 的核心是一个允许你采用简洁的模板语法来声明式的将数据渲染进 DOM 的系统。
结合响应系统，在应用状态改变时， Vue 能够智能地计算出重新渲染组件的最小代价并应用到 DOM 操作上。

条件语句

v-if

v-if可根据表达式的值来渲染内容

v-if可搭配v-else、以及v-else-if使用，但这两个必须紧跟在v-if后使用，否则不会被识别

v-show

大致用法同v-if，与v-if不同的是，带有v-show的 元素始终保留在DOM中，它只是简单的切换元素的display属性

如果非常频繁地切换，则使用 v-show 较好；如果在运行时条件很少改变，则使用 v-if 较好

监听器

当需要在数据变化时执行异步或开销较大的操作时，使用监听器是最有用的。

事件处理

事件处理可以使用v-on指令，通常情况下，我们需要使用一个方法来调用 JavaScript 方法。v-on可以接收一个定义的方法来调用

事件修饰符

修饰符是由点开头的指令后缀来表示的
.stop：阻止冒泡
.prevent：阻止默认事件
.capture：阻止捕获
.self：只监听触发该元素的事件
.once：只触发一次
.passive：不拦截默认事件

按键修饰符

在监听键盘事件时可添加按键修饰符

表单输入绑定

可使用v-model 指令在表单控件元素上进行双向数据绑定，v-model回忽略表单元素的初始值，如果想要设置初始值，则需要在组件的data选项中声明初始值，对于不同的表单元素，绑定的类型不同
文本框、多行文本：绑定值为字符串
复选框：单个复选框绑定到布尔值，多个复选框绑定到同一个数组
